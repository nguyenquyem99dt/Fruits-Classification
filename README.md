# Fruits Classification
1. Run demo.py by command line<br>
  - Step 1: Open cmd <br>
  - Step 2:
    + If input is image:
      Type: python demo.py --model=model.h5 --input=<path to image> --image=True
    + If input is video:
      Type: python demo.py --model=model.h5 --input=<path to video>
2. Dataset: fruit-360
